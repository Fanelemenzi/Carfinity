Configuration Guide

How to Configure Authentication Groups:

  1. Create Django Group: First create a Django group in the Groups section
  2. Add Authentication Config: Create an AuthenticationGroup configuration linking to the Django group
  3. Set Dashboard: Configure which dashboard the group provides access to
  4. Set Priority: Higher priority groups become the default when users have multiple groups
  5. Organization Compatibility: Specify which organization types should automatically get this group

Dashboard Types:

  • Customer: For regular customers accessing vehicle and maintenance info
  • Insurance: For insurance company users accessing policies and risk data
  • Admin: For administrative users with full system access
  • Technician: For service technicians accessing maintenance tools
  • Custom: For custom dashboard implementations
