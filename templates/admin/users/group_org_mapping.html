Recommendations

Best Practices:

  • Link organizations to groups: Ensure each organization is linked to appropriate groups so members automatically get correct permissions
  • Customer organizations should be linked to the "customers" group
  • Insurance organizations should be linked to the "insurance_company" group
  • Review members without groups: These users may not have proper dashboard access
  • Use bulk management: Fix multiple users at once using the bulk management tool
